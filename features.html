Features

Everything ConTexture does, in detail.

The writing environment

A clean, distraction-free editor that gets out of your way — with everything you need always within reach.

✍️

Markdown editor

Full Markdown support including headings, bold, italic, underline, inline code, code blocks, quote blocks, bullet lists, numbered lists, and task checkboxes.

🔤

Typography controls

Choose from Palatino, New York, Georgia, System, or Monospace. Adjust font size, line spacing, and column width to your preference.

👁️

Live preview

Switch between edit and preview modes instantly. Preview renders full Markdown including images, headings, and formatting.

🔗

Wiki links

Type [[ to link any note to any other note in your project. Navigate your world like a wiki.

@

@Mentions

Type @ to reference images, character sheets, or other notes inline. Images inserted as Markdown render in preview.

💬

Inline comments

Select any text and add a comment anchored to it. Comments appear in the margin panel alongside your writing.

Project organisation

Three workspaces keep different kinds of writing cleanly separated.

📖

Creative Writing

Longform fiction projects with manuscript chapters, character sheets, outline folders, maps and images, and research notes — all nested under a single project.

📅

Journal

Daily writing space with calendar integration. See your events and reminders alongside your daily note. Great for morning pages, meeting notes, and reflections.

📚

References

A flat reference library for anything that doesn't belong in a project — writing notes, style guides, research clippings, and saved quotes.

🧑‍🎨

Character sheets

Structured character profiles with photo, biography, physical description, personality, relationships, and freeform markdown notes. Link characters to each other.

🗺️

Maps & images

Dedicated image folders for world maps, reference photos, and visual inspiration. Import from your photo library and reference images in your writing with @mentions.

📌

Pinned notes

Pin your most-used notes for quick access from anywhere in the app. Bookmarks panel keeps your most important references at hand.

The margin panel

Five tabs of tools that live beside your writing without interrupting it.

Aa

Format tab

Bold, italic, underline, headings 1–2, bullet, numbered, task, wiki link, indent, outdent, centre, inline code, code block, quote block — all with keyboard shortcuts.

≡

Outline tab

Auto-generated document outline from your headings. Jump to any section instantly.

💬

Comments tab

All comments on the current document in one scrollable list. Click any comment to jump to its anchor in the text.

#

Footnotes tab

Insert and manage footnotes inline. Footnotes are stored as Markdown-compatible syntax.

🖼️

Images tab

Images associated with the current document shown in a gallery. Tap to preview or insert into the editor.

📊

Stats tab

Word count, character count, estimated reading time, and page count for the current document.

Sharing & collaboration

Share your work in the way that makes sense for who you're sharing with.

👥

Live collaboration

Share a note via iCloud file sharing. Both people edit the same .md file live — works with any Markdown editor, no ConTexture required on the other side.

📤

Rich export

Export a note as a rich .md file with YAML front matter and comments preserved as footnotes. Another ConTexture user can import it with full fidelity.

📝

Plain Markdown

Export a clean .md file that opens perfectly in Obsidian, iA Writer, VS Code, Bear, or any other Markdown app.

📥

Import .md files

Import any .md or .markdown file from anywhere on your device or iCloud via File → Import. Choose which project and folder it lands in.
